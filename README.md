# Pet API - Documentação

Este projeto utiliza as seguintes ferramentas

- `express.js 4.19.2`
- `pg 8.11.5`
- `pg-hstore 2.3.4`
- `sequelize 6.37.3`
- `sequelize-cli 6.6.2`
- `prettier 3.2.5`
- `Docker 24.0.5`

Requisitos para o ambiente de desenvolvimento

- `Composer`
- `npm`
- `Git`
- `Docker Engine (ou Docker Desktop)`
- `Prettier (npm)`

Clone o repositório através do comando

- `git clone https://github.com/martinezrafael/pet-api.git`

Instalar dependências

- Execute o comando `npm install`

Instalar o Postgres via Docker Compose
